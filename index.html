Covid-19 SQL-queries

This project showcases SQL proficiency by querying a COVID-19 database to extract and transform data for Tableau visualization. Key techniques included aggregate functions (SUM, COUNT) with GROUP BY clauses, JOIN operations to combine infection data with population tables for percentage calculations, window functions for time-series analysis, and potentially CTEs for complex multi-step transformations. By pulling data from dimension tables (geography, dates, countries) and fact tables (infections, deaths, population), the queries created denormalized datasets optimized for Tableau, demonstrating essential ETL processes and data aggregation skills for business intelligence and analytics.

Covid-19 Dashboard

This Tableau dashboard delivers a comprehensive analysis of COVID-19 from 2020-2021, leveraging SQL-queried data to present key insights through four interconnected visualizations. It features a global summary scorecard highlighting total cases, deaths, and death rate percentage, alongside a geographical heat map illustrating infection rates by country. A horizontal bar chart identifies continental death tolls, with Europe most impacted, while a multi-line temporal chart tracks infection trends across countries, distinguishing actual versus estimated values. The dashboard demonstrates advanced Tableau capabilities and effective data storytelling, transforming complex pandemic data into actionable visual insights.
